# UE ローカリゼーションツール

- `.csv` のライブラリを置き換えました。
- **Google Sheets** と **Crowdin** へのインポート時にスペースが失われる原因となっていた `.csv` の処理を​​修正しました。
- 大きな `.locres` を保存する際にプログラムがフリーズする問題を修正しました。
- 非同期メソッドを使用することで、プログラムを高速化しました。
- ウクライナ語化とインターフェースの改善を行いました。
- `.locres v4` (**Stellar Blade**) をサポートしました。
